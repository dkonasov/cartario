---

## What I do

- read specified story from `docs/project-overview.md`
- break it down into ready for implementation tasks
- create a section for the plan for implementing the story in `docs/plan.md` with checkboxes for each task

## When to use me

Use me when you need to create a plan for user story. DON'T use me when you need to implement a user story, use `implement-user-story` skill for that.
